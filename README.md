# blog-archive

A static archive of [flow14](https://www.flow14.com), my personal blog, which ran from 2006 to 2014 on WordPress. I exported it with [Simply Static](https://simplystatic.com/) and host it as a read-only snapshot to preserve the content.

Many external links and embedded media are broken — that's expected. All posts and comments are intact.

## Structure

Posts are organized by year (`/2006` through `/2014`) with category, tag, and author index pages. CSS and JS live in `/css` and `/js`.

## Maintenance

Used `cleanup.py` to strip dead WordPress and Google Analytics artifacts from the HTML files.
